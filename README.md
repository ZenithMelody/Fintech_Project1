# 🏦 Risk Loan Prediciton deployed as web application with Logistic Regression (with Scikit-Learn)

## ✨ What does it do?
* **Note:** This project ≠ replace analyst, but a filter, tells the analyst where to spend their limited time + energy to protect institution’s capital

## 🚩 Issues & Solution
* **Reading:** My lack of background in business, inner workings and their terminologies.
* **Fix:** I had to use AI to breakdown concepts, so some business terms may not be 100% but I tried.

## 🛠️ Tech Stack
* **Language:** Python 3.13, JS
* **UI:** HTML, CSS
* **Library:** 
  * `scikit-learn`
  * `pandas` and `numpy` (data manipulation)
* **Algorithm:** Logistic Regression
